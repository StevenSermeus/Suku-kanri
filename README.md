# Suku kanri

The goal of this project is to create a task management system that fit my need as a student and a developer.

## Features

- [ ] Create a task
  - [ ] Title
  - [ ] Description
  - [ ] Due date
  - [ ] Priority
  - [ ] Tags
- [ ] Edit a task
- [ ] Delete a task
  - [ ] Don't notify the user
- [ ] Notification for a task
- [ ] Filter tasks
  - [ ] By due date
  - [ ] By priority
  - [ ] By tags
  - [ ] By status
